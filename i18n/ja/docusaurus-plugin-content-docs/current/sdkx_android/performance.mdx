# インパクト分析 {#performance}

<Intro>

以下の番号は、Helpshift SDK と統合された枠組みとなるサンプルアプリ
を示しています。

</Intro>

## v10.3.0

### アプリ内の SDK ペイロード {#payload-size}

Helpshift SDK X は、アプリケーションのリリース APK に`231 Kb`を追加します。
この分析は、Android Gradle Plugin `v3.4.2`と Gradle `v5.3`を使用して実行されました。
`231 Kb`というサイズには、すべてのリソースとコードを含む影響の合計値が含まれています。

### メソッドの数 {#methods-count}

Helpshift SDK X は、あなたの dex に 1373 件のメソッドを追加します。

| パッケージ        | メソッドの数 |
| ----------------- | -----------: |
| **com.helpshift** |       `1373` |

### メモリの使用量 {#memory-usage}

| Helpshift API の起動           | Java ヒープメモリ（単位: MB） | 総メモリ使用量（単位: MB） |
| ------------------------------ | ----------------------------- | -------------------------: |
| showFAQs                       | `4.65`                        |                  `102-110` |
| showFAQSection                 | `4.65`                        |                    `80-90` |
| showSingleFAQ                  | `3.55`                        |                    `55-70` |
| showConversation（会話モード） | `6.05`                        |                  `142-148` |

このメモリへの影響は、アプリが実行されているデバイスの OS およびメモリ構成にも依存します。

<Admonition type="info" title="注意">

Android 14 を搭載した Google Pixel 6 上のデモアプリでテストされています。

</Admonition>

| パッケージ        | メソッドの数 |
| ----------------- | -----------: |

### メモリの使用量 {#memory-usage}

`Helpshift.showConversation (Myactivity.this, configMap)` API を呼び出して Helpshift の会話画面を開くと、
約 20 - 25MB のメモリが消費されます。
このメモリへの影響は、アプリが実行されているデバイスの OS およびメモリ構成にも依存します。

<Admonition type="info" title="注意">

Android 10 が搭載された Samsung Galaxy M11 上のデモアプリでテストされています。

</Admonition>
