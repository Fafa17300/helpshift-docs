# インパクト分析 {#performance}

<Intro>

SDK のリリースごとに、このページでは SDK がアプリケーションのビルドと実行時間のさまざまな側面に与える影響を追跡します。

</Intro>

<Admonition type="info" title="注意">

**アプリの起動時間**

Helpshift の初期化はすべてバックグラウンドのスレッドで実行されるため、Helpshift SDK がアプリの起動時間に影響を与えることはありません。

</Admonition>

# バージョン別のレポート {#version-impact}

Helpshift SDK の各バージョンのインパクト分析レポートについては、以下でご確認いただけます。

## 10.3.0

<Admonition type="info" title="注意">

macOS 13.5.1、Xcode 15.0.0、iOS 17.0.3（iPhone 15 Plus）の環境で実施しています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

xcframework が占有するディスク容量です。

xcframework のサイズは、デバイスが 1.2MB、シミュレーターが 2MB です。合計サイズは 3.2MB です。

内訳は以下の通りです。

| アーキテクチャ | デバイスのサイズ（MB） | シミュレーターのサイズ（MB） |
| -------------- | ---------------------- | ---------------------------- |
| arm64          | 0.85                   | 0.83                         |
| x86_64         | NA                     | 0.77                         |

iOS アプリが`HelpshiftX.xcframework`と統合された場合、IPA ファイルサイズの増加は約`366`KB です。

### RAM の使用量

|                                                                     |                                                                                   |
| ------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **Helpshift のチャット画面（<code>showConversation:</code>）**      | <code>3.7</code>MB が追加されます（チャット画面上のメッセージの数に依存します）。 |
| **Helpshift の FAQ 画面（<code>showFAQs:</code>）**                 | <code>2.8</code>MB が追加されます。                                               |
| **Helpshift の FAQ セクション画面（<code>showFAQSection:</code>）** | <code>4.1</code>MB が追加されます。                                               |
| **Helpshift の単一の FAQ 画面（<code>showFAQ:</code>）**            | <code>2.3</code>MB が追加されます。                                               |
