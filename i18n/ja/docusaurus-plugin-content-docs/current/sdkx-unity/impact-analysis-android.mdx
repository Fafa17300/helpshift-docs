# インパクト分析 {#performance}

<Intro>

以下の番号は、Helpshift Unity プラグインと統合された枠組みとなるサンプルアプリ
を示しています。

</Intro>

<Admonition type="info" title="注意">

**アプリの起動時間**

Helpshift の初期化はすべてバックグラウンドのスレッドで実行されるため、Helpshift SDK がアプリの起動時間に影響を与えることはありません。

</Admonition>

## v10.3.0

### アプリ内の SDK ペイロード {#payload-size}

Helpshift Android プラグインは、アプリケーションのリリース APK に約`250 Kb`を追加します。
`250 Kb`というサイズには、すべてのリソースとコードを含む影響の合計値が含まれています。
SDK X に appcompat ライブラリを使用すると、約`1MB`が追加されます。

### メソッドの数 {#methods-count}

Helpshift SDK X は、あなたの dex に 1404 件のメソッドを追加します。

| パッケージ        | メソッドの数 |
| ----------------- | -----------: |
| **com.helpshift** |       `1404` |

### メモリの使用量 {#memory-usage}

| Helpshift API の起動           | Java ヒープメモリ（単位: MB） | 総メモリ使用量（単位: MB） |
| ------------------------------ | ----------------------------- | -------------------------: |
| showFAQs                       | `4.6`                         |                   `95-105` |
| showFAQSection                 | `4.30`                        |                    `75-90` |
| showSingleFAQ                  | `3.80`                        |                    `45-65` |
| showConversation（会話モード） | `5.50`                        |                      `135` |

このメモリへの影響は、アプリが実行されているデバイスの OS およびメモリ構成にも依存します。

<Admonition type="info" title="注意">

Android 11 を搭載した Google Pixel 4a 上のデモアプリでテストされています。

</Admonition>

## v1.0.0-beta​ {#integration-impact-100-beta}

### アプリ内の SDK ペイロード {#payload-size-100-beta}

Helpshift Android プラグインは、アプリケーションのリリース APK に`92`KB を追加します。92KB というサイズには、すべてのリソースとコードを含む影響の合計値が含まれています。SDK X に appcompat ライブラリを使用すると、約 1MB が追加されます。

### メソッドの数 {#methods-count-100-beta}

Helpshift SDK X は、あなたの dex に 518 件のメソッドを追加します。

| パッケージ        | メソッドの数 |
| ----------------- | -----------: |
| **com.helpshift** |        `518` |

### メモリの使用量 {#memory-usage-100}

`HelpshiftSdk.GetInstance().ShowConversation(configMap)` API を呼び出して Helpshift の会話画面を開くと、約 20 - 25MB のメモリが消費されます。このメモリへの影響は、アプリが実行されているデバイスの OS およびメモリ構成にも依存します。

<Admonition type="info" title="注意">

Android 10 が搭載された Samsung Galaxy M11 上のデモアプリでテストされています。

</Admonition>
