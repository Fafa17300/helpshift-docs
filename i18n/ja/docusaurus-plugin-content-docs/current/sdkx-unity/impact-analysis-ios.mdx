# インパクト分析 {#performance}

<Intro>

SDKのリリースごとに、このページではSDKがアプリケーションのビルドと実行時間のさまざまな側面に与える影響を追跡します。

</Intro>

<Admonition type="info" title="注意">

**アプリの起動時間**

Helpshiftの初期化はすべてバックグラウンドのスレッドで実行されるため、Helpshift SDKがアプリの起動時間に影響を与えることはありません。

</Admonition>

# バージョン別のレポート {#version-impact}

Helpshift SDKの各バージョンのインパクト分析レポートについては、以下でご確認いただけます。

## 10.1.0

<Admonition type="info" title="注意">

Unity 2019.4.11f1、macOS 12.1、Xcode 13.2の環境でビルドしたデモアプリをiOS 15.1（iPhone X）でテストしています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

Unity iOSアプリが`HelpshiftX.framework`と統合された場合、IPAファイルサイズの増加は約`240`KBです。

### RAMの使用量

以下は、デバッグモードでのHelpshift APIのメモリ使用量です。（値はおおよそのものです）

|                                                                   |                                                                                    |
| ----------------------------------------------------------------- | ---------------------------------------------------------------------------------- |
| **Helpshiftのチャット画面（<code>showConversation:</code>）**      | <code>10.3</code>MBが追加されます（チャット画面上のメッセージの数に依存します）。 |
| **HelpshiftのFAQ画面（<code>showFAQs:</code>）**              | <code>10.1</code>MBが追加されます。                                                         |
| **HelpshiftのFAQセクション画面（<code>showFAQSection:</code>）** | <code>10</code>MBが追加されます。                                                           |
| **Helpshiftの単一のFAQ画面（<code>showFAQ:</code>）**         | <code>10</code>MBが追加されます。                                                           |

## 10.0.0

<Admonition type="info" title="注意">

Unity 2019.4.11f1、macOS 10.15.7、Xcode 12.0の環境でビルドしたデモアプリをiOS 14.0（iPhone X）でテストしています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

Unity iOSアプリが`HelpshiftX.framework`と統合された場合、IPAファイルサイズの増加は約`200`KBです。

### RAMの使用量

以下は、デバッグモードでのHelpshift APIのメモリ使用量です。（値はおおよそのものです）

|                                                                   |                                                                                   |
| ----------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **Helpshiftのチャット画面（<code>showConversation:</code>）**      | <code>8.6</code>MBが追加されます（チャット画面上のメッセージの数に依存します）。 |
| **HelpshiftのFAQ画面（<code>showFAQs:</code>）**              | <code>8.3</code>MBが追加されます。                                                         |
| **HelpshiftのFAQセクション画面（<code>showFAQSection:</code>）** | <code>8.1</code>MBが追加されます。                                                         |
| **Helpshiftの単一のFAQ画面（<code>showFAQ:</code>）**         | <code>8</code>MBが追加されます。                                                           |

## 1.0.0-beta

<Admonition type="info" title="注意">

Unity 2019.4.11f1、OS X 10.15.6、Xcode 11の環境でビルドしたデモアプリをiOS 12.4.5（iPhone 6 Plus）でテストしています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

Dynamic Frameworkが占有するディスク容量です。

フレームワークの合計サイズは`1.7 MB`です。

内訳は以下の通りです。

| アーキテクチャ | サイズ（MB） |
| ------------ | --------- |
| arm64        | 0.421     |
| armv7        | 0.373     |
| i386         | 0.368     |
| x86_64       | 0.372     |

Unity iOSアプリがHelpshiftXフレームワークと統合された場合、IPAファイルサイズの増加はフレームワークのビットコード版と非ビットコード版の両方について約`140 KB`KBです。

### RAMの使用量

メモリの使用量（デバッグモードでは、値はおおよそのものです。）

|                                                              |                                                                                   |
| ------------------------------------------------------------ | --------------------------------------------------------------------------------- |
| **Helpshiftのチャット画面（<code>showConversation:</code>）** | <code>4.5</code>MBが追加されます（チャット画面上のメッセージの数に依存します）。 |
