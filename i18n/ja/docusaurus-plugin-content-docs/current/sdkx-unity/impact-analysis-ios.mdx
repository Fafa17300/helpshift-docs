# インパクト分析 {#performance}

<Intro>

SDK のリリースごとに、このページでは SDK がアプリケーションのビルドと実行時間のさまざまな側面に与える影響を追跡します。

</Intro>

<Admonition type="info" title="注意">

**アプリの起動時間**

Helpshift の初期化はすべてバックグラウンドのスレッドで実行されるため、Helpshift SDK がアプリの起動時間に影響を与えることはありません。

</Admonition>

# バージョン別のレポート {#version-impact}

Helpshift SDK の各バージョンのインパクト分析レポートについては、以下でご確認いただけます。

## 10.3.0

<Admonition type="info" title="注意">

Unity 2021.3.14f1、macOS 13.5.1、Xcode 15.1 の環境でビルドしたデモアプリを iOS 17.2（iPhone 15 Plus）でテストしています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

Unity iOS アプリが`HelpshiftX.framework`と統合された場合、IPA ファイルサイズの増加量は約`369`KB です。

### RAM の使用量

以下は、デバッグモードでの Helpshift API のメモリ使用量です。（値はおおよそのものです）

|                                                         |                           |
| ------------------------------------------------------- | ------------------------- |
| **Helpshift のチャット画面（`ShowConversation`）**      | `10.3`MB が追加されます。 |
| **Helpshift の FAQ 画面（`ShowFAQs`）**                 | `6.1`MB が追加されます。  |
| **Helpshift の FAQ セクション画面（`ShowFAQSection`）** | `7.3`MB が追加されます。  |
| **Helpshift の単一の FAQ 画面（`ShowFAQ`）**            | `8.8`MB が追加されます。  |

## 1.0.0-beta

<Admonition type="info" title="注意">

Unity 2019.4.11f1、OS X 10.15.6、Xcode 11 の環境でビルドしたデモアプリを iOS 12.4.5（iPhone 6 Plus）でテストしています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

Dynamic Framework が占有するディスク容量です。

フレームワークの合計サイズは`1.7 MB`です。

内訳は以下の通りです。

| アーキテクチャ | サイズ（MB） |
| -------------- | ------------ |
| arm64          | 0.421        |
| armv7          | 0.373        |
| i386           | 0.368        |
| x86_64         | 0.372        |

Unity iOS アプリが HelpshiftX フレームワークと統合された場合、IPA ファイルサイズの増加はフレームワークのビットコード版と非ビットコード版の両方について約`140 KB`KB です。
